# Legna.cn 网站分析与排期

## 概览

- **域名**：legna.cn
- **定位**：纯净无广告的应用导航，面向电脑小白和装机用户
- **技术栈**：Vue 3 + Vite + Element Plus + Pinia + Vue Router + Axios

## 网站结构

```
legna.cn/
├── 首页（应用分类 + 搜索 + 应用卡片）
├── 应用详情页（介绍 + 下载 + 安装说明）
├── LegnaOS 页面（系统介绍 + 一键安装）
└── 关于 / 反馈
```

## 优化建议

| 方向 | 问题 | 建议 |
|------|------|------|
| SEO | SPA 爬虫抓不到内容 | 引入 SSR（Nuxt）或预渲染 |
| 性能 | 图片未优化 | WebP + CDN + gzip |
| 无障碍 | 禁止了用户缩放 | 移除 `user-scalable=no` |
| 体验 | 无离线支持 | 添加 PWA + Service Worker |
